Artwork Guidelines

Artwork File Guidelines for Custom Packaging

Send print-ready files in AI, PDF, CDR or PSD with 3 mm bleed, CMYK color mode, embedded fonts (or outlined), 300 dpi minimum images, and separate foil/embossing layers. Full checklist below.

Preferred File Formats

  • Adobe Illustrator (.AI) 鈥?best. All text outlined, all raster images embedded.
  • PDF/X-1a or PDF/X-4 鈥?flattened, CMYK, 3 mm bleed, embedded fonts.
  • CorelDRAW (.CDR) 鈥?accepted, save down to X6 for compatibility.
  • Photoshop (.PSD) 鈥?accepted for continuous-tone artwork, 300 dpi, CMYK, with layer named "Foil" / "Emboss" if needed.

Bleed & Safe Area

  • Bleed: 3 mm around all cut lines
  • Safe area: 3 mm inside all cut and fold lines for text and logo
  • Crop marks: included, offset 3 mm from trim

Color Mode

  • CMYK 4-color offset for full-color print
  • Pantone Solid Coated for spot color (specify Pantone number in file legend)
  • Rich black recipe: C60 M40 Y40 K100 (avoid 100% K only for large solid areas)

Images & Resolution

  • Raster image minimum 300 dpi at final print size
  • Vector artwork preferred for logos, illustrations and pattern
  • Embed all linked images, do not link externally

Fonts

  • Outline all fonts before sending (Illustrator: Type 鈫?Create Outlines)
  • Or send the font files together with the artwork if outlining is not possible

Foil / Embossing / Debossing / Spot UV Layers

  • Use a separate spot color named "Foil" (or "Emboss" / "Deboss" / "SpotUV") set to Overprint
  • Fill zones 100% solid, no gradient
  • Provide a written note stating: gold / silver / rose gold / holographic foil

Barcode & GS1 DataMatrix

  • Barcode: EAN-13 / UPC-A / Code 128 as vector, min height 25.93 mm, magnification 80鈥?00%
  • GS1 DataMatrix: vector, min cell size 0.375 mm, quiet zone 1 cell around
  • Do not resize or distort a rendered barcode image; supply as vector or barcode data

Dieline

  • Dieline stroke width 0.25 pt in spot color "Dieline"
  • Cut lines: solid
  • Fold lines: dashed
  • Perforation lines: dotted
  • Dieline on a separate layer above artwork, non-printing

File Naming & Delivery

Preferred name: BrandName_ProductName_Qty_Date.ai. Upload via WeTransfer, Google Drive, Dropbox or email the file to Linda Wang at linda@colorprintingpackage.com. Files above 25 MB should use a cloud link.
